Core Commands

keypair generate
Generate and manage Stellar keypairs securely. Easily create, fund, and display account keys.
$ chainctl keypair generate
balance
Check XLM and asset balances for any Stellar account address on Testnet or Mainnet.
$ chainctl balance GABCD...
send
Transfer assets between accounts seamlessly. Supports custom assets and memo fields.
$ chainctl send --to GXYZ... --amount 10
contract deploy
Deploy and invoke Soroban smart contracts directly from your workspace.
$ chainctl contract deploy ./lib.wasm
network use
Switch seamlessly between public, testnet, futurenet, and local networks.
$ chainctl network use testnet
export payments
Export account history, payments, and trades into structured formats like CSV.
$ chainctl export payments --csv

Why chainctl?

JSON Output by Default

Easily pipe command output to jq or other CLI tools for automated workflows and scripting.

Testnet and Mainnet Support

Built-in configuration for Mainnet, Testnet, Futurenet and local sandbox environments.

Soroban Ready

First-class support for compiling, deploying, and invoking Stellar smart contracts.

CSV Export Support

Extract complex historical data like payments, operations, and trades into simple CSV files.
